# homebrew-tools

Homebrew tap for [Aurelian-Shuttleworth](https://github.com/Aurelian-Shuttleworth) projects.

## Install agent-ear

```bash
brew tap Aurelian-Shuttleworth/tools
brew install agent-ear
```

For full documentation, see the [agent-ear repository](https://github.com/Aurelian-Shuttleworth/agent-ear).

## Available Formulas

| Formula | Description |
|:--------|:------------|
| `agent-ear` | Agentic voice capture — agent-steerable audio transcription |
